# 🚀 AI ExamNotes

AI ExamNotes is a full-stack AI-powered web application that generates structured, exam-oriented notes instantly using Google Gemini API.

It helps students create high-quality revision notes, diagrams, charts, and downloadable PDFs in seconds.

---

## 🌟 Features

✅ Google Authentication (Firebase)  
✅ AI-Powered Exam Notes (Gemini API)  
✅ Structured & High-Scoring Content  
✅ Flowcharts (Mermaid Diagrams)  
✅ Comparison Tables & Charts  
✅ PDF Download Support  
✅ Credit-Based System  
✅ Responsive & Modern UI  
✅ Secure Backend (JWT + Cookies)  
✅ MongoDB Database Integration  

---

## 🧠 How It Works

1. User logs in with Google
2. User selects:
   - Topic
   - Class Level
   - Exam Type
   - Revision Mode
   - Include Diagrams / Charts
3. Backend sends structured prompt to Gemini API
4. AI generates high-quality markdown notes
5. Notes are saved in MongoDB
